### Quick Start

```bash
# Clone the repo
git clone https://github.com/clintmccoy/hermes.git
cd hermes

# Install dependencies
npm install

# Set up environment variables
cp .env.example .env.local
# Fill in .env.local with your values

# Run development server
npm run dev
```
